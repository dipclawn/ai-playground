## ✨ 功能

| 功能 | 说明 |
|------|------|
| 💬 **多模型聊天** | 支持100+模型，流式输出，Markdown渲染 |
| 🎨 **AI图片生成** | 文生图，支持多种分辨率和模型 |
| 💭 **模型浏览** | 浏览所有可用模型，一键复制ID |
| ⚙️ **配置自由** | 可切换任意模型，自带演示Key |

## 🔧 技术栈

- **SiliconFlow API** — 国内大模型聚合平台
- **Streaming API** — 流式输出响应
- **PWA** — 可安装到桌面，离线可用
- **纯前端** — 所有API调用直接走浏览器

## 🚀 使用

1. 打开 https://dipclawn.github.io/ai-playground/
2. 点击右上角 ⚙️ 填入你的 SiliconFlow API Key
3. 选择模型，开始聊天或生成图片

> 💡 获取 API Key: https://platform.siliconflow.cn/
